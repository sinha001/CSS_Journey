  • Em

  • Rem (Root Em)

  • vh

  • vw

    relative to 1% viewport width.


2. Position:

    The position CSS property sets how an element is positioned in a document.
    Example:
    position: static/relative/absolute/fixed.

  • static:

    default position(The top, right, bottom, left, and z-index properties have no effect).


  • relative:

    element is relative to itself.(The top, right, bottom, left, and z-index will work).


  • absolute:

    position relative to its closet positioned ancestor.(removed from z-index).


  • fixed:

    position relative to browser.(removed from flow).


  • sticky:

    position based on user's scroll position.

  • z-index:

    It decides the stack level of elements.
    Overlapping elements with a larger z-index cover those with a smaller one.

    • z-index: auto (0)
    • z-index: 1 / 2 / ..
    • z-index: -1 / -2 / ...

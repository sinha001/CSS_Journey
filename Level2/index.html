CONTENT
  • Height

    By default, it sets the content area height of the element.

    Example: height: 50px;


  • Width

    By default, it sets the content area width of the element.

    Example: width: 50px;


  • Border

    Used to set an element's border.

    Example: border-width: 2px; border-style: solid/dotted/dashed; border-color: black; border: 2px dotted black; border-radius: 10px; //or 50%


  • Padding

    Example: padding-left:5px; padding-right:50px; padding-top:5px; padding-bottom:50px;
    padding-shorthand.. padding:25px; padding:1px 2px 3px 4px;


  • Margin

    Example: margin-top:50px; margin-bottom:50px; margin-left:50px; margin-right:50px;
    margin shorthand... margin:50px; margin:5px 0px 5px 0px;

3. Display Property

display: inline/block/inline-block/none
  • inline:

    Takes only the space required by the element. (no margin/padding).


  • block:

    Takes full space available in width.


  • inline-block:

    Similar to inline, but we can set margin & padding.


  • none:

    To remove element from document flow.


4. Visibility

visibility: hidden

Note: When visibility is det to none, space for the element is reserved.
But for display set to none, no space is reservedor blocked for the element.

5. Alpha Channel

opacity(0 to 1)
  • RGBA:

    color: rgba(255,0,0,0.5);
    color: rgba(255,0,0,1);
    0 -- means not visible.
    1 -- means completely visible.
